# games

Each approved game gets its own folder here, generated by the daily build pipeline:

```
games/
  <game-id>/
    index.jsx     ← exports default React component for the game
    spec.json     ← copy of the game spec used to build it
    assets/       ← optional sprites, audio, etc.
```

Games are imported lazily by the router and the arcade machine entries in `world.js`.

Use `_template/` as the starting point for new games.
